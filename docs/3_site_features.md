# Altered Vault Format Website Features

The Altered Vault Format website is for player registration, card pool tracking, and season management. It does not handle gameplay. All features focus on fairness, transparency, and ease of use.

## Pages and Features

### Home Page

- Overview of the Altered Vault Format and current season.
- Display active sets.
- "Register / Log in" entry point for players.

### Player Dashboard

- Shows the player’s current pool, broken down by set.
- Displays booster count (e.g., 24/36 used).
- Allows entry of booster packs (via scan or manual entry).
- Shows rarity breakdown and total boosters per set.
- Displays past season history.

### Player Directory

- List of all registered players in the current season.
- Shows each player’s total boosters and sets included.
- Clicking a player shows their full pool (viewable only to logged-in users).

### Season Management (Admin Tools)

- Define current season, active sets, and booster limits.
- Approve or remove players.
- Manage rotation (retiring old sets, adding new ones).
- Publish announcements and updates on the homepage.

### Rules & Format Page

- Official Altered Vault Format rules and FAQ.
- Clarifies how sealed integration works, how boosters are added, and rotation timing.
- Displays past rule versions (for historical reference).

### Optional Features

- Guided “Add Booster” flow for entering packs.
- Statistics for rarity distribution and total cards.
- Player stats and progress visuals.
- Export of player pools for verification.

## Privacy and Access

- Site is semi-private: only registered and logged-in users can view other players’ pools.
- Admins have full visibility and control over seasons and data.
